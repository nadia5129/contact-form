
### Project Overview

This project is my full-stack portfolio application built for SDEV 305. I designed it to function as my professional website, combining my resume, project portfolio, and a fully functional contact system into one cohesive application.

The site includes a responsive resume page, a portfolio showcasing both web development and Game Boy Advance projects, and a contact form that connects to a MySQL database. I also implemented a secure admin dashboard.
